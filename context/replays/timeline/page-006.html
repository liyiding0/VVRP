Move VVRP interface IPv4 ownership into IP module

Moved VVRP interface IPv4 address storage out of IFNET and into IP state. IFNET now keeps only interface admin/MAC/MTU/protocol state, while imported interface snapshots are assembled with IP-owned addresses for consumers such as RM, ARP, ICMP, and FWD. IP address changes update IFNET protocol state explicitly instead of IFNET deriving protocol state from addresses.
